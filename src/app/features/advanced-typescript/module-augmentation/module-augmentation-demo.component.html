Module Augmentation & Declaration Merging

Extend existing types and modules without modifying their source code.

What is Module Augmentation?

Module augmentation and declaration merging allow you to extend existing types, interfaces, and modules without modifying their original source code. This is especially useful for extending third-party libraries.

Key Concepts

  • Module Augmentation: Extend types from external modules
  • Declaration Merging: Merge multiple declarations of the same name
  • Global Augmentation: Extend global types like Array, Window, etc.
  • Namespace Merging: Merge namespaces with interfaces/classes
