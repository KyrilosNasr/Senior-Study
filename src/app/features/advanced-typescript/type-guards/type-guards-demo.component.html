Narrow types at runtime and enable type-safe code with type guards and predicates.

What are Type Guards?

Type guards are functions that help TypeScript narrow down types at runtime. They use type predicates (value is Type) to tell the compiler that a value is of a specific type after a runtime check.

Key Concepts

  • Type Guards: Functions that narrow types using value is Type syntax
  • Type Predicates: Return type annotations that tell TypeScript about type narrowing
  • Discriminated Unions: Union types with a common property used for type narrowing
