What are Decorators?

Decorators are a special kind of declaration that can be attached to classes, methods, properties, and parameters. They use the form @expression where expression evaluates to a function.

Types of Decorators:

  • Class Decorators: Applied to class declarations
  • Method Decorators: Applied to method declarations
  • Property Decorators: Applied to property declarations
  • Parameter Decorators: Applied to parameter declarations

Key Takeaways

  • ✓ Decorators are functions that modify classes, methods, properties, or parameters
  • ✓ Decorator factories return decorator functions and accept parameters
  • ✓ Enable powerful patterns like dependency injection, logging, and validation
  • ✓ Angular uses decorators extensively (@Component, @Injectable, etc.)
