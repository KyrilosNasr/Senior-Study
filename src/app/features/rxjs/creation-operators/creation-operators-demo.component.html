Creation Operators

Creating observables from different sources: of(), from(), and fromEvent()

What are Creation Operators?

Creation operators are functions that create observables from various sources. They're the foundation of reactive programming in Angular.

Key Operators

  • of(): Emits values synchronously, then completes - Use for static data, config, mock data
  • from(): Converts arrays, promises, iterables to observables - Use for async conversions
  • fromEvent(): DOM events to observable streams - Use for user interactions, events

Key Takeaways

  • Use of() for static values, config, or mock data
  • Use from() to convert promises, arrays, or iterables to observables
  • Use fromEvent() for DOM events - makes event handling reactive and cancellable
  • Creation operators are the foundation - they turn any data source into a reactive stream
