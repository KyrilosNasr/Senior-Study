RxJS WebSocket

WebSocketSubject provides a reactive interface to WebSocket connections with built-in reconnection logic, error handling, and composability with RxJS operators.

Features

  • Automatic reconnection with exponential backoff
  • Observable-based API
  • Error handling and recovery
  • Composable with RxJS operators
